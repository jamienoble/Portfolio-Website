# Hero Images Folder

Place your hero banner images in this folder.

## Recommended Image Specifications:
- **Format**: JPG or PNG
- **Dimensions**: 1920x1080px or larger (16:9 aspect ratio)
- **File size**: Under 500KB for optimal loading
- **Name**: Use descriptive names like `hero-1.jpg`, `hero-2.jpg`, etc.

## Usage:
The hero banner will automatically use any images you place here. You can add multiple images and they'll be displayed with a dark overlay for text readability.

## Example filenames:
- hero-1.jpg
- hero-2.jpg
- creative-team.jpg
- office-workspace.jpg
